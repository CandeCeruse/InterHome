1. Instalar Python (Flask solo soporta Python 3.8 o posterior).

4. Una vez que estamos en el entorno (debe encontrarse un (.venv) detras del nombre del equipo tal que; (.venv) [Nombre]@raspberrypi:):
    $ pip install Flask
